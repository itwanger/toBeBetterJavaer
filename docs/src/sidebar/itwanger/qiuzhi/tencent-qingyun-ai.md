## 05、青云计划怎么投？

青云计划覆盖六大技术方向：大语言模型、NLP（自然语言处理）、多模态、智能体（Agent）、搜索推荐、AI 基础设施。

不管你是搞模型训练的，还是做 Agent 应用落地的，还是搞底层 infra 的，都有岗位。

投递入口，浏览器地址栏输入 `join.qq.com`，选择青云计划，选择你擅长的方向，点击【实习生报名】，投递简历就可以了。

![](https://cdn.paicoding.com/tobebetterjavaer/images/mdnice/35d1c2e2cf7c-9d7c0cdb-8fca-46ac-beb5-b56e6db30c39.jpg)

全是最热门的课题方向，就比如说这个微信搜索-大模型 AI 搜索研究与落地。

![](https://cdn.paicoding.com/stutymore/tencent-qingyun-ai-20260419124422.png)

顺带给大家分享一下，2026 年我们应该重视的核心竞争力，那就是理解系统的复杂性、抽象和结构化思维、驾驭不确定性。

![](https://cdn.paicoding.com/tobebetterjavaer/images/mdnice/dcfc011efdc2-a9df0d79-127a-4115-a4df-fdfaec40fc57.jpg)

这三项能力，也是 Harness Engineering 的底层能力，可以让我们在 AI Agent 时代构建起专属于我们个人的护城河：

- **理解系统的复杂性**——知道一个 Agent 系统里有哪些模块，每个模块在什么时机做什么事儿，模块之间怎么通信、怎么传递状态。
- **抽象和结构化思维**——能从一堆乱七八糟的需求里，抽出最核心的循环、最本质的工具边界、最关键的上下文结构。
- **驾驭不确定性**——能在一个概率性的系统里，设计出确定性的兜底逻辑。这是最难的，因为传统软件工程训练的是「一切都要可预测」，而 Agent 系统本质上是「大部分时候可预测、少部分时候彻底失控」。

## 06、为什么有鹅选鹅

聊聊我自己的看法。

第一，平台决定了你能接触到什么级别的数据、什么规模的算力、什么水平的同事。

就比如说前文提到的 Hy3 preview， 2026 年 1 月底开始训练，今天就发布了，短短不到三个月，就有这样瞩目的成绩，离不开鹅厂足够的算力和人力啊。

只能说，腾讯这个数量级的互联网大厂，实力能与之匹敌的真的屈指可数。

第二，整个公司的文化是技术导向的，在这样的环境里，我们可以有时间有精力去做自己喜欢的事情。

特别适合作为职业生涯的“第一站”。有从 0 到 1 的培养体系，有足够大的场景让你练手，有顶级的同事共同成长。底子打好了，后面不管是留下还是出去闯，都会从容得多。

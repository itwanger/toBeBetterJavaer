## 01、检查你的微信版本

首先，微信需要升级到最新版。

目前只能是 iPhone，但安卓显然不会等太久。

升级路径：微信 → 我 → 设置 → 关于微信 → 检查新版本。

需要升级到 **8.0.70** 版本。

升级完成后，杀掉微信进程，重新打开微信。

然后进入：我 → 设置 → 插件。

如果能看到「微信 ClawBot」，就说明你已经灰度到了。

![](https://cdn.paicoding.com/stutymore/sucai-20260322123457.png)

点击详情，就可以看到接入方法了。

## 02、安装微信 OpenClaw CLI

接入非常简单，在电脑上执行下面的命令：

```bash
npx -y @tencent-weixin/openclaw-weixin-cli@latest install
```

![](https://cdn.paicoding.com/stutymore/sucai-20260322124052.png)

这个命令会从 npm 下载腾讯官方的微信 OpenClaw CLI 工具，自动完成安装配置。

命令执行后，CLI 工具会自动完成以下事情：

1. 下载并安装微信 OpenClaw 适配器
2. 检查本地 OpenClaw 环境
3. 生成微信连接配置
4. 启动本地 Gateway 服务

稍等片刻，就会看到微信的扫码界面了。

![](https://cdn.paicoding.com/stutymore/sucai-20260322124250.png)

这个二维码是微信官方生成的临时连接凭证，有效期大概 5 分钟。如果超时了，会重新给你生成一个新的二维码。

拿起微信，扫码，就可以连接了。

![](https://cdn.paicoding.com/stutymore/sucai-b8c857b2d7d2564c4f5b1c68ad1cc4ee.jpg)

扫码成功后，微信会提示已连接 OpenClaw。

整个流程非常顺畅，不需要配置飞书，不需要注册企业微信，就是个人微信直接接入。

这可是真正的国民级用户啊。

## 04、和飞书接入的对比

用过飞书接入 OpenClaw 的小伙伴应该知道，飞书接入的流程是这样的：

**飞书接入流程：**

1. 注册飞书开放平台账号
2. 创建企业自建应用，填写应用名称和描述
3. 添加机器人能力，获取 App ID 和 App Secret
4. 配置权限管理，开通 `im:message`、`im:message:send_as_bot` 等权限
5. 配置事件订阅，开启长链接模式，添加 `im.message.receive_v1` 事件
6. 在 OpenClaw 中配置飞书通道，填入 App ID 和 App Secret
7. 发布应用版本，等待飞书审核（通常秒过）
8. 在飞书里找到机器人，发送消息测试

整个过程大概需要 5-10 分钟。

**微信接入流程：**

1. 升级微信到 8.0.70
2. 检查是否灰度到 ClawBot 插件
3. 执行一行命令：`npx -y @tencent-weixin/openclaw-weixin-cli@latest install`
4. 扫码连接

全程 5 分钟搞定，而且是个人微信直接可用。

当然，微信接入目前还是灰度测试阶段，功能可能不如飞书接入那么完整。比如飞书支持的多维表格操作、日程管理、审批流程等高级功能，微信这边暂时还没有。

但从体验上来说，个人微信直接能用这件事，本身就是巨大的进步。它把 OpenClaw 的门槛从企业级应用降到了个人玩具，让更多人能体验到 AI Agent 的能力。

而且微信的社交属性是飞书无法比拟的。你的 OpenClaw 可以直接出现在微信聊天列表里，和你的家人、朋友、同事在同一个界面。这种无缝的体验，是飞书做不到的。

## 05、使用场景畅想

微信接入 OpenClaw 之后，能干什么？

我最想要的场景包括。

**场景一：个人知识管理**

平时在微信里看到好文章，直接转发给 ClawBot，让它帮你提取关键信息、生成摘要、分类归档。再也不用收藏夹吃灰了。

![](https://cdn.paicoding.com/stutymore/wechat-openclaw-20260322131404.png)

虽然 OpenClaw 直接获取不了内容，但是已经可以做到调用浏览器打开网页内容。

并把内容发给我了。

![](https://cdn.paicoding.com/stutymore/wechat-openclaw-20260322131526.png)

**场景二：群聊智能助手**

在微信群聊里，@ClawBot 让它帮你干活。比如：

- @ClawBot 总结一下今天的讨论要点
- @ClawBot 把刚才的文件转 PDF 发给我
- @ClawBot 提醒我今天下午 3 点开会

ClawBot 会直接在群里回复，所有人都能看到结果。

**场景三：文件处理中心**

微信的文件传输助手可以和 OpenClaw 联动。发一个文件过去，Agent 自动帮你处理：

- 压缩包 → 自动解压、列出文件清单
- Excel 表格 → 数据分析、生成图表
- Word 文档 → 提取大纲、生成摘要
- PDF 文件 → OCR 识别、文字提取

操作方式：文件传输助手 → 发送文件 → ClawBot 自动处理 → 返回结果。

想象一下这个场景：

你在地铁上，微信上收到一个压缩包，里面是一堆 Excel 表格。你把文件发给 ClawBot，让它帮你合并、去重、生成汇总报表。等你到公司，报表已经做好了，直接下载就行。

这种体验，以前只能在电脑上完成，现在手机上就能搞定。

这就是微信接入 OpenClaw 的意义——让 AI 助手真正融入日常沟通场景，而不是一个需要专门打开的应用。
